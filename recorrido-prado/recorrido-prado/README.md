# Distrito Candelaria - Recorrido Prado (sitio estático)

Sitio *one-page* (diurno) sobre el recorrido por el barrio Prado, preparado para publicarse en GitHub Pages.

## Contenido incluido
- index.html
- css/styles.css
- js/main.js
- images/ (imágenes de ejemplo — reemplaza con fotos propias)

## Cómo publicar en GitHub Pages (pasos rápidos)
1. Crea un repositorio nuevo en GitHub (por ejemplo: `distrito-candelaria`).
2. Sube todos los archivos y carpetas a la rama `main`.
3. Ve a Settings → Pages, selecciona la rama `main` y la carpeta `/ (root)`; guarda.
4. GitHub proveerá la URL `https://<tu-usuario>.github.io/<repo>/` en unos minutos.

## Notas
- Reemplaza las imágenes en `images/` por tus fotos reales (mismas rutas y nombres).
- Inserta tu iframe de Google Maps en la sección de mapa.
- Para recibir inscripciones reales integra Formspree, Google Forms o un backend.
